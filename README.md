#  Smart Expense Tracker 

An offline-first expense tracker with **custom-built analytics using pure JavaScript and HTML Canvas**.  

##  Features
- Add & delete expenses
- Category-based tracking
- Persistent data using `localStorage`
- Bar chart (category-wise expenses)
- Line graph (daily spending)
- Analytics built from scratch using Canvas API

---

##  Tech Stack
- HTML
- CSS
- Vanilla JavaScript (ES6)
- HTML Canvas
- localStorage

---

## folder Structure
index.html # Expense input
